# Agents

This file documents the agents and tools used in this project.

## Development Agents

### Lint
Command: `npm run lint`
Description: Runs linting to check code quality

### Typecheck
Command: `npm run typecheck`
Description: Runs TypeScript type checking

## Build Agents

### Build
Command: `npm run build`
Description: Builds the project for production

### Test
Command: `npm test`
Description: Runs the test suite

## Game Features

### Core Mechanics
- Click system with critical hits
- Auto-clicker upgrade system
- Egg purchasing with discounts
- Pet hatching and merging
- Upgrade progression (7 types)
- Achievement system (14 achievements)
- Save/load system

### Content
- 14 egg types with unique drop pools
- 50+ pets across 8 rarity tiers
- Complex upgrade trees (8 upgrade types)
- Achievement tracking
- Statistics system
- Multi-hatch egg purchasing
